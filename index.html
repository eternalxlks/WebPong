Hacker's Console

Controls: Player A: W and S. Player B: Up and Down Arrow Key! (You MUST full screen this tab!)

Change Score
scoreA = 1; // Changes Score of Player A
scoreB = 1; // Changes Score of Player B
Change Paddle Properties
// You can replace "paddleA" with "paddleB"
// in these examples.
paddleA.height = 10; // Change height
paddleA.width = 10; // Change width
paddleA.x = 10; // Change x position
paddleA.y = 10; // Change y position
Change the Ball's Properties
x = 10; // x position
y = 10; // y position
dx = 5; // horizontal speed 
dy = 5; // vertical speed
Useful Constants & Functions
canvas // HTML canvas element
canvas.width // Canvas width
canvas.height // Canvas height
